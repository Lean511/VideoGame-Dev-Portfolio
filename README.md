# VideoGame-Dev Portfolio

 ¡Hola! Soy Leandro Friedrich, estudiante de programación enfocado en el desarrollo de sistemas y mecánicas para videojuegos.
 Este repositorio contiene una selección de mis proyectos técnicos y prototipos de práctica.
 Estos proyectos son juegos con el principal objetivo de practicar y aprender a desarrollar en Unity, por lo que no estan enfocados en ser una experiencia
 real y completa.

 Dentro de cada carpeta de cada juego se encuentra un ejecutable para probarlos, archivos involucrados en el desarrollo de los juegos 
 y un README con más información sobre cada uno.

## 🚀 Proyectos Destacados

### 1. Ascencio (Ajedrez)
*   **Descripción:** Un juego de ajedrez funcional con validación de movimientos y lógica de turnos.
*    Este es un prototipo en el que se planea trabajar más adelante. El concepto es hacer un juego clásico de Ajedrez combinado con mecánicas modernas
*    donde cada pieza va a poseer distintas habilidades únicas que harían el gameplay más variado y llamativo.
*    Mi papel fue el diseñar la infraestructura básica del código para que todo el sistema funcione, además de la declaración y funcionalidad de 
*    cada pieza del tablero.

### 2. Proyecto Plataformero
*   **Descripción:** Proyecto de práctica enfocado en el control del personaje y sistemas de colisiones.
* 	 Este es un prototipo básico enfocado en los sistemas de movimiento del personaje. Este puede interactuar con el entorno (con recogibles, 
*    enemigos o el suelo)
*	 La propuesta del juego es que el jugador aumente su velocidad de movimiento mientras no se detenga. Llegada a cierta velocidad, es capáz de eliminar
*    a los enemigos.
*    Todo el código del juego está desarrollado por mi.

### 3. Space Shooters
*   **Descripción:** Proyecto de práctica enfocado en mecánicas de movimiento 3D y Raycast.
*	 Este prototipo es simplemente un juego donde el jugador controla una nave y debe disparar a las naves enemigas. Contiene un sistema de puntuación y
*    de spawn aleatorio de naves enemigas.
*    Todo el código del juego está desarrollado por mi.
